
### SWEA 사전학습 과제 track3

`D1 : 1936. 1대1 가위바위보`

A와 B가 가위바위보를 하였다.

가위는 1, 바위는 2, 보는 3으로 표현되며 A와 B가 무엇을 냈는지 입력으로 주어진다.

A와 B중에 누가 이겼는지 판별해보자. 단, 비기는 경우는 없다.

**[입력]**

입력으로 A와 B가 무엇을 냈는지 빈 칸을 사이로 주어진다.

**[출력]**

A가 이기면 A, B가 이기면 B를 출력한다.

```python
A, B = input().split()    ##숫자를 띄어쓰기로 넣으면 나눠서 구분됨
A = int(A)
B = int(B)
result = ""

if A == 1:
    if B == 2:
        print('B')
    elif B == 3:
        print('A')

if A == 2 :
    if B == 1:
        print('A')
    elif B == 3:
        print('B')

if A == 3:
    if B == 1:
        print('B')
    elif B == 2:
        print('A')

print(result)
```
